## Getting Started
Make sure you have already installed jekyll on your system. Please install first.

`gem install bundler jekyll`

# install dependencies
1.  `npm install`

# serve with hot reload at localhost:8080
2.  `npm run dev || npm run serve`

# build for deploy
3. `npm run build`
